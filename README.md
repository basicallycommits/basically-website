- Dark mode support

## Setup Instructions

1. Clone the repository:
   ```sh
   git clone [https://github.com/yourusername/my-website.git](https://github.com/basicallycommits/basically-website.git)
   cd my-website
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
